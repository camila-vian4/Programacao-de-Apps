# Programacao-de-Apps

## Comandos Git Básicos

- **`git init`**  
  Inicializa um repositório Git local na pasta atual.

- **`git add .`**  
  Adiciona todas as alterações (arquivos novos, modificações e exclusões).

- **`git commit -m "mensagem"`**  
  Registra as alterações adicionadas com uma mensagem descritiva.

- **`git push -u origin main`**  
  Envia os commits para o repositório remoto na branch `main` (define o upstream na primeira vez).
